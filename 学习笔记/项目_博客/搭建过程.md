# 博客搭建笔记

## Hexo博客框架

```shell
1.安装Node.js
    地址:nodejs.org
安装检查
	cmd窗口检查 
> node -v 
> npm -v
出现版本号=》node.js安装成功

2.安装git
   https://blog.csdn.net/weixin_44107920/article/details/108631178


2.安装一个国内比较快的源
> npm install -g cnpm --registry=https://registry.npm.taobao.org
安装检查
	cnpm -v
	
3.安装hexo框架
> cnpm install -g hexo-cli
安装检查
	hexo -v
	
4.建立一个博客文件夹 blog

5.进入你的blog初始化hexo
> sudo hexo init
	当出现 start blogging with Hexo! 成功了
	ls -l   // 这些都是初始化的
    2022/04/13  19:47    <DIR>          .github
    2022/04/13  19:47                89 .gitignore
    2022/04/13  19:48    <DIR>          node_modules
    2022/04/13  19:48           168,224 package-lock.json
    2022/04/13  19:47               641 package.json
    2022/04/13  19:47    <DIR>          scaffolds
    2022/04/13  19:47    <DIR>          source
    2022/04/13  19:47    <DIR>          themes
    2022/04/13  19:47                 0 _config.landscape.yml
    2022/04/13  19:47             2,546 _config.yml

6.启动博客
> hexo s
localhost:4000


7.新建博客
hexo new "文件名"
在source/_posts 里面就会建立md文件

8.回到blog下，更新  <可以先清理一下  hexo clear>
> hexo g       
```

## 部署到github

```
1.首先你的有一个gayhub账号
2.头像旁边的+号 New repository
设置你的Repository地址：必须是  用户名.github.io
Desciption随便写个描述 就可以了

3.安装git插件
> cnpm install --save hexo-deployer-git

4.配置 _config.yml 到文件的最底部
# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: https://github.com/nairongIT/nairongIT.github.io.git
  branch: master
  
5.部署到github
> git config --global user.email "nairongIT@163.com"
> git config --global user.name "nairongIT"
> hexo d
```
